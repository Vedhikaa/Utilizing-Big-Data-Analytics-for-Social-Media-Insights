✈️ Airline Sentiment Analysis using Machine Learning






🚀 Overview

This project uses machine learning and natural language processing (NLP) to analyze airline-related tweets and classify them into Positive, Negative, or Neutral sentiments. It helps uncover customer perceptions and evaluate airline service quality through data-driven insights.

📌 Features

Text preprocessing (tokenization, stopword removal, lemmatization)

TF-IDF feature extraction

Multiple ML models: Logistic Regression, Naive Bayes, Random Forest

Model evaluation using Accuracy, Precision, Recall, and F1-score

Sentiment visualization using charts and word clouds

📂 Dataset

Airline tweet dataset containing labeled customer feedback with sentiment categories: Positive, Negative, and Neutral.

🛠 Installation

Clone the repository:

git clone https://github.com/yourusername/airline-sentiment-analysis.git
cd airline-sentiment-analysis

Install dependencies:

pip install -r requirements.txt

Run the model:

python train_model.py
